Smart Tools
Cargo Placement
Smart Bounce NEW
При перекрытии грузов после drag — контейнер мягко отталкивается в ближайшую свободную позицию рядом, а не остаётся в overlap.
Smart Grid Snap NEW
При сбросе контейнера рядом с соседом или границей бэя — один раз аккуратно выравнивает по ближайшей логичной позиции. Не липнет при следующем движении.
Keyboard Shortcuts NEW
Del=удалить, R=повернуть, D=дублировать, E=редактировать, 1–5=zoom. Нажмите ? для полного списка.
Highlight by Platform
When enabled, clicking a platform dims all unrelated cargo. When off, all cargo stays fully visible.
Weight Gauge
Shows total deck weight vs 2500T max capacity as a fill bar in the header.
Empty Deck Hint
Shows a placement hint when no cargo is on deck.
Show DG Only
Hides all non-DG cargo blocks for quick compliance review.
Cargo Library
Manifest Matching NEW
Показывает расхождения между импортированным ASCO списком и реальным расположением груза на палубе.
Safety
DG Auto-Segregation Check SAFETY
При размещении DG груза — автоматическая проверка совместимости по матрице IMDG. Нарушения сегрегации подсвечиваются немедленно.
Visual
DG Badge Fade on Hover NEW
При наведении на грузовой блок — DG и HL бейджи становятся полупрозрачными, чтобы был виден CCU/ID.
Cargo Hover Motion NEW
При наведении на грузовой блок он слегка приподнимается и увеличивается. Выключите для полностью статичного интерфейса.
